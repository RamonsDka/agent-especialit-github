# Roadmap

## 1.0.0 Baseline

- Portable agent and composite skill package.
- Explicit Task-only invocation with deny-by-default local mutation.
- `gh`-first evidence collection and traceability contract.
- Native bounded review alignment and deterministic lens documentation.
- Installation, security, workflow, output, and contribution documentation.

## Possible Follow-Ups

- Add repository-owned automated frontmatter and JSONC linting without making
  the runtime agent dependent on a new tool.
- Add redaction-focused fixtures for example output validation.
- Add a maintainer-approved compatibility matrix for OpenCode releases.
- Add verified adapters only after their provenance and permission model are
  documented; do not make MCP configuration implicit.
